# **App Name**: SwiftMove

## Core Features:

- Service Display: Display available porter services, pricing, and real-time availability through a user-friendly interface.
- Booking Form: Provide a straightforward booking form for customers to schedule porter services.
- Authentication: Secure user and admin authentication system with login/register functionality.
- Admin Panel Access: Role-based routing to ensure admins have access to a protected admin panel.
- Admin Management: Admin panel to view bookings, manage porters (CRUD operations), view users, and update booking status in real-time.
- Smart Destination Suggestion: Uses a tool to learn common destinations based on the current and past bookings to automatically suggest destinations to users to simplify the booking process and increase conversion.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to evoke trust and efficiency.
- Background color: White (#FFFFFF), offering a clean and modern backdrop.
- Accent color: Royal blue (#4169E1) for key CTAs and interactive elements.
- Body and headline font: 'Roboto' sans-serif for a modern and objective feel.
- Use clear, simple icons from 'react-icons' to represent services, availability, and user actions.
- Employ a clean, responsive layout using Next.js App Router and Tailwind CSS, ensuring optimal viewing experience on all devices.
- Subtle transitions and animations using 'shadcn/ui' to enhance user interaction and provide feedback on actions.
- Implement a typewriter effect for displaying key information or welcome messages to add a modern and engaging touch.
